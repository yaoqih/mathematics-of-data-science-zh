# 翻译规范

## 基本原则

- 忠实传达数学含义，中文表达以自然、准确、简洁为先。
- 公式、变量、定理编号、引用键和原始 `label` 不翻译。
- 人名定理首次出现时使用“中文名（英文名）”，后文使用统一中文名。
- 专有术语优先采用 `glossary/terms.yml`；新译法先更新术语表。
- 不擅自修正可能影响结论的内容。疑似错误用 issue 记录，并注明原文位置。
- 图片中的英文不直接覆盖；需要汉化时保留原图并新增派生文件。

## 章节状态

每章 front matter 使用下列状态之一：

- `待翻译`：只有章节骨架；
- `翻译中`：正文尚未完整；
- `待校对`：已完成初译；
- `已校对`：完成数学和语言校对；
- `已发布`：满足授权和发布质量要求。

## 标记约定

```markdown
::: {.callout-note title="译者注"}
仅用于解释必要的语境，不引入未经标识的新结论。
:::
```

交叉引用使用 Quarto 标识，例如 `@sec-introduction`、`@thm-example`。参考文献沿用原始 BibTeX citation key。
